# Code of Conduct

LawLink 社区讨论应围绕问题本身，保持专业、具体、可验证。

## 基本原则

- 尊重不同经验水平的使用者和贡献者。
- 批评代码、方案和事实，不攻击个人。
- 不公开他人的真实案件资料、个人信息、联系方式、密钥或内部文件。
- 不用项目 issue 或讨论区提供具体个案法律意见。
- 对安全问题使用私密报告渠道，不在公开区贴漏洞利用细节。

## 不接受的行为

- 人身攻击、骚扰、歧视性表达。
- 泄露或索取真实案件材料、客户信息、身份证号、手机号、住址、统一社会信用代码等敏感数据。
- 发布恶意代码、漏洞利用脚本、密钥或绕过鉴权的步骤。
- 重复刷屏、偏离项目主题的推广内容。

## 执行方式

维护者可以编辑、隐藏或删除违反本准则的内容，也可以临时或永久限制相关账号参与项目讨论。

本准则用于维护项目协作环境，不替代 GitHub 平台规则和适用法律。
